Projects

I'll be posting projects I've worked on and am working on from time to time on my github, but I'll also summarize a couple interesting ones here.

Correlation in Fantasy Football (Master's Thesis)

  • Obtained game data from 2009-14 from nfl.com using Andrew Gallant's nflgame API and formatted it for analysis in R

  • Developed basic parametric assumptions concerning the estimation of fantasy point correlations of NFL players by position, which allowed for inference on these correlations

  • Analyzed and demonstrated how the results can be utilized in both season-long and weekly fantasy leagues

Classifying Handwritten Digits

  • Implemented a number of algorithms in both R and Python to classify digits in the MNIST handwritten digits data set

  • Algorithms included: PCA, k-means, Fisher's discriminant analysis, logistic regression, SGD, SVM, and mixture models (via the EM algorithm)

  • Achieved 9 % test error rate using SVM and 11 % using logistic regression

Binary Star System Classification

  • Analyzed 24 GB of Hubble Telescope light flux data for over 150,000 star systems using PySpark

  • Fit a non-parametric regression to each star system's light curve using Gaussian and Epanechnikov kernels (bandwidth selected via LOOCV) to produce residual light readings

  • Identified potential binary star systems by detecting systematic trends in the residual light readings
